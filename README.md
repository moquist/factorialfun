# factorialfun

Demos of stack-consuming recursion and the 'recur special form to calculate factorials.

## Usage Examples

```bash
$ lein run f1 4
$ lein run f2 4
$ lein run f1 10000 # BOOM! Stack [probably] blows here.
$ lein run f2 10000
```
